✅ **Google 字体** – 使用优雅的排版增强可读性。

# 📥 安装与使用  

## 💻 **1. 克隆仓库**  
要开始使用，您需要将项目从 GitHub 克隆到本地机器。打开终端或命令提示符并运行：  

```sh
git clone https://github.com/passerby-png/NovaX-Genesis.git
```

这将在您的计算机上创建仓库的本地副本。  

## 📂 **2. 打开项目**  
完成后，导航到项目文件夹：  

```sh
cd NovaX-Genesis
```

现在，只需在浏览器中打开 `index.html` 文件即可查看项目。您也可以在 VS Code 中使用 **Live Server** 获得更好的开发体验。  

## 🔧 **3. 自定义和修改**  
此项目设计为 **易于自定义**。  

- 在 `index.html` 中修改 **HTML 结构**。  
- 在 `style.css` 中调整样式或添加您自己的自定义样式。  
- 在 `script.js` 中编辑 JavaScript 动画和交互。  
- 用您自己的资产替换图像和图标。  

---

# 🛠️ 如何使用

1️⃣ **设置您的 NFT 列表**  
- 用您自己的资产替换 `assets/images` 文件夹中的占位符 NFT 图像。  
- 在 `index.html` 中修改 NFT 标题、描述和价格。  

2️⃣ **自定义配色方案和 UI**  
- 打开 `style.css` 并修改颜色变量以匹配您的品牌。  
- 根据需要调整字体大小和 UI 间距。  

3️⃣ **增强动画**  
- 在 `script.js` 中修改 GSAP 动画以更改效果、速度和行为。  
- 调整 **ScrollTrigger** 设置以创建独特的基于滚动的效果。  

4️⃣ **优化性能**  
- 压缩 CSS 和 JS 文件以加快加载时间。  
- 如有需要，为图像使用 **延迟加载**。
